Politique de Confidentialité Introduction Bienvenue sur PETITE CAISSE. La confidentialité de vos informations personnelles est essentielle pour nous. Cette Politique de Confidentialité décrit comment nous collectons, utilisons et protégeons vos informations lorsque vous utilisez notre application. En téléchargeant ou en utilisant PETITE CAISSE, vous acceptez les pratiques décrites dans cette Politique. Informations que nous collectons Nous collectons les types de données suivants : Informations personnelles : Nom et prénom (si fournis). Numéro de téléphone ou adresse e-mail (lors de l'inscription). Données relatives à l'utilisation de l'application : Rapports générés ou informations saisies via l'application. Données de connexion et d'identification : Adresse IP et informations sur l'appareil (si nécessaire pour le dépannage). Utilisation des données collectées Les informations collectées via l'application sont utilisées pour : Fournir et améliorer les services de l'application. Générer des rapports personnalisés que vous pouvez consulter ou télécharger. Gérer les données via Google Firestore, un service sécurisé de Google Cloud, qui nous permet de stocker et gérer vos informations de manière fiable. Partage des données Nous ne partageons pas vos informations personnelles avec des tiers, sauf dans les cas suivants : Si nécessaire pour respecter une obligation légale. Pour résoudre des problèmes techniques spécifiques liés à l'application, avec votre consentement explicite. Stockage et sécurité des données Où vos données sont-elles stockées ? Vos données personnelles et rapports sont sauvegardés sur : Votre appareil (pour les rapports générés localement). Google Firestore, qui garantit des mesures de sécurité avancées. Comment protégeons-nous vos données ? Nous utilisons des protocoles de sécurité tels que l'authentification Firebase pour protéger vos données. Vos données sont cryptées lors du transfert et stockées en toute sécurité dans Firestore. Vos droits Vous avez les droits suivants : Accéder à vos données : Vous pouvez consulter vos données via l'application. Modifier ou supprimer vos données : Vous pouvez demander la suppression de vos données en nous contactant. Pour exercer ces droits, contactez-nous à l’adresse e-mail indiquée ci-dessous. Modifications de cette Politique de Confidentialité Nous pouvons mettre à jour cette Politique pour refléter les modifications apportées à notre application ou pour répondre à des exigences légales. La version la plus récente sera toujours disponible via un lien public.
